# Contribuindo para o Calculadora de Juros Simples

Obrigado por se interessar em contribuir! Este projeto faz parte de uma iniciativa de microfinanças para empoderar indivíduos de baixa renda.

## Como posso ajudar?

### 1. Relatando Bugs
Se você encontrar algum erro no cálculo ou no código:
* Verifique se o erro já foi relatado nas **Issues**.
* Caso não, abra uma nova Issue descrevendo o problema, o comportamento esperado e passos para reproduzir o erro.

### 2. Sugerindo Melhorias
Ideias para novas fórmulas financeiras ou melhorias no script são bem-vindas! Abra uma Issue com a tag `enhancement`.

### 3. Enviando Alterações (Pull Requests)
1. Faça um **Fork** do repositório.
2. Crie uma branch para sua modificação: `git checkout -b minha-melhoria`.
3. Faça o **Commit** das suas alterações: `git commit -m 'Adiciona nova funcionalidade'`.
4. Faça o **Push** para a branch: `git push origin minha-melhoria`.
5. Abra um **Pull Request** para a branch principal.

## Padrões de Código
* Mantenha o código simples e comentado.
* Siga as boas práticas da linguagem utilizada.
